## Welcome

This is the documentation home for the **Nexis ecosystem** — a family of
developer tools built around the Nexis terminal: two ML engines, a local-AI
benchmark app, a multi-repo dev dashboard, and a shared design system. Pick a
starting point below, or use the search box (<kbd>/</kbd>) to jump straight to
a topic.
