Shortcuts below adapt to your operating system automatically — use the keyboard
picker in the top bar to switch between Linux, macOS, and Windows layouts.

## Tabs & panes

| Action | Shortcut |
| --- | --- |
| New terminal tab | <Kbd linux="ctrl+t" mac="cmd+t" windows="ctrl+t" /> |
| New private terminal | <Kbd linux="ctrl+r" mac="cmd+r" windows="ctrl+r" /> |
| New editor tab | <Kbd linux="ctrl+e" mac="cmd+e" windows="ctrl+e" /> |
| Close tab / pane | <Kbd linux="ctrl+w" mac="cmd+w" windows="ctrl+w" /> |
| Next tab | <Kbd linux="ctrl+tab" mac="ctrl+tab" windows="ctrl+tab" /> |
| Previous tab | <Kbd linux="ctrl+shift+tab" mac="ctrl+shift+tab" windows="ctrl+shift+tab" /> |
| Jump to tab 1–9 | <Kbd linux="ctrl+1" mac="cmd+1" windows="ctrl+1" /> … <Kbd linux="ctrl+9" mac="cmd+9" windows="ctrl+9" /> |
| Split pane right | <Kbd linux="ctrl+d" mac="cmd+d" windows="ctrl+d" /> |
| Split pane down | <Kbd linux="ctrl+shift+d" mac="cmd+shift+d" windows="ctrl+shift+d" /> |
| Focus next pane | <Kbd linux="ctrl+]" mac="cmd+]" windows="ctrl+]" /> |

## Search & navigation

| Action | Shortcut |
| --- | --- |
| Find in terminal | <Kbd linux="ctrl+f" mac="cmd+f" windows="ctrl+f" /> |
| Shell history search | <Kbd linux="ctrl+r" mac="ctrl+r" windows="ctrl+r" /> |
| Quick file open | <Kbd linux="ctrl+p" mac="cmd+p" windows="ctrl+p" /> |
| Search in files | <Kbd linux="ctrl+shift+f" mac="cmd+shift+f" windows="ctrl+shift+f" /> |
| Command palette | <Kbd linux="ctrl+shift+p" mac="cmd+shift+p" windows="ctrl+shift+p" /> |

## Editing & code

| Action | Shortcut |
| --- | --- |
| Format document | <Kbd linux="shift+alt+f" mac="shift+alt+f" windows="shift+alt+f" /> |
| Rename symbol | <Kbd linux="f2" mac="f2" windows="f2" /> |
| Refactor (LSP code actions) | <Kbd linux="ctrl+shift+r" mac="cmd+shift+r" windows="ctrl+shift+r" /> |
| Fold all | <Kbd linux="ctrl+k ctrl+0" mac="cmd+k cmd+0" windows="ctrl+k ctrl+0" /> |
| Unfold all | <Kbd linux="ctrl+k ctrl+j" mac="cmd+k cmd+j" windows="ctrl+k ctrl+j" /> |
